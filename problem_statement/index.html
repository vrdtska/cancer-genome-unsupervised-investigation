Introduction

Problem Statement

Glioblastoma (GBM) is a highly aggressive brain tumor characterized by severe molecular heterogeneity. This biological variance often renders universal treatment protocols ineffective, as different transcriptomic subtypes respond differently to therapies. The challenge lies in navigating high-dimensional bulk RNA-sequencing (RNA-seq) data to uncover these hidden phenotypes. The objective of this project is to architect an unsupervised machine learning pipeline capable of parsing a ~60,000-feature transcriptomic dataset to mathematically isolate distinct Glioblastoma subtypes and extract the specific genetic biomarkers driving their molecular divergence. The genomic data utilized in this study is sourced from The Cancer Genome Atlas (TCGA) program and was accessed via the UCSC Xena multi-omics data hubs. Furthermore, gene expression was quantified using the STAR alignment pipeline and normalized to Transcripts Per Million (TPM), ensuring robust transcript abundance comparisons across patient samples.

Methodology

To address the problem of dimensionality inherent in genomic data, the pipeline reduced feature space from biomarker discovery. First, variance filtering isolated the top 10,000 highly variable genes, which were subsequently standardized and compressed via Principal Component Analysis (PCA). The scree plot empirically justified retaining a lower-dimensional subspace capturing 80% of the variance. Next, a dual-metric heuristic sweep (Silhouette Score and Davies-Bouldin Index) evaluated K-Means against Agglomerative Hierarchical clustering. Hierarchical clustering with Ward's linkage proved mathematically superior, successfully mapping the clonal evolution of the tumors and identifying an optimal structural partition of K=4. Finally, to extract actionable biological meaning, the boundaries of the two dominant subtypes (Subtype 0 and Subtype 1) were mapped back to the full ~60,000-feature log-normalized dataset. A non-parametric Mann-Whitney U differential expression analysis, visualized via a Volcano Plot, successfully decoupled background noise from true genomic signal, isolating the specific upregulated and downregulated proteins defining each cohort. The Gemini large language model (Google, 2026) was utilized during the development of this project to assist with code optimization as well as data pipeline structuring.
